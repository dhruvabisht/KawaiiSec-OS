## 🌸 animefetch.sh

Your adorable system info fetcher.

### 📂 Location:
```bash
resources/scripts/animefetch.sh
```

### ▶️ Run it:

```bash
./resources/scripts/animefetch.sh
```

### 📋 What it shows:

* Current user, host, uptime, OS, shell
* RAM usage on macOS (via vm_stat)
